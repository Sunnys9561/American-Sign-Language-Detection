# 🤟 American Sign Language Detection

##  Overview
This project uses Deep Learning (CNN) to detect American Sign Language hand gestures.

##  Features
- Classifies 29 classes (A–Z + SPACE + DELETE + NOTHING)
- Built using TensorFlow & Keras
- Trained on ASL dataset

##  Model
- Convolutional Neural Network (CNN)
- Achieves ~90%+ accuracy
